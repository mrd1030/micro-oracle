# Micro Oracle

**One genuinely useful micro-action, right now.**

## Advanced Features
- Rate actions with 1-5 stars
- Add notes when logging wins
- Weekly summary + charts
- Streak protection options

## Capacitor Native Build

```bash
# Install
npm install @capacitor/core @capacitor/cli

# Initialize
npx cap init "Micro Oracle" "com.mrd1030.microoracle"

# Add platforms
npx cap add ios
npx cap add android

# Sync web assets (make sure www/ has your built files)
npx cap sync

# Open native projects
npx cap open ios
npx cap open android
```

Then build and submit to the App Store and Play Store.
